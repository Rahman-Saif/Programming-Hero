css position

  1. Static:eita hocche default position behaviour!.Usual text or something jei position e thake setai static positioning
  2. Relative: eitate div er nijer position ta bojay thake..tobe eitake dane bame neya jay..but eitar jei poriman height or width ache seita ekdom same to same setai thakbe
  3. Absolute:eitate nijer kono position or obosthan thakbe na..sojha bangly er payer tola nai..eitake jar mon chay tar upore boshay deya jay jemon ekta div er position jodi absolute koira dei,taile tar height,width thakbe but website e kono place dokhol korbe na,tokhon tar jaygay nicher div ta aeisa porte parbe..ar jehetu eitar kono tola nai website e ,tai eitake onno jekono image or div er upor boshano jay
  4. Fixed:ei bandare jodi konokhane rakhte chai taile ekdom same to same oi jaygatei boisha jay
  5. Sticky: ei banda fixed er motoi ,jeikhane boshaite chamu oikhanei boisha jaibo.but difference hocche eita tar parent er size onujayi kaj kore..i mean parent er height or width er baire jabe na!
